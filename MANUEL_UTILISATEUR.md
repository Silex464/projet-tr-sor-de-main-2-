## 1. Présentation du site

**Trésor de Main** est une plateforme en ligne dédiée à la mise en relation entre artisans et amateurs d'art artisanal. Le site permet :

- Aux **acheteurs** de découvrir et d'ajouter en favoris des créations uniques
- Aux **artisans** de présenter et vendre leurs œuvres fait-main

### Types de créations disponibles :
- Bijoux
- Sculptures
- Peintures
- Céramique
- Textile
- Travail du bois
- Verrerie
- Maroquinerie (cuir)
- Métallerie

## 3. Créer un compte

### Étape 1 : Accéder à l'inscription

1. Cliquez sur **"Connexion"** dans la barre de navigation
2. Cliquez sur le lien **"Créer un compte"** sous le formulaire de connexion

### Étape 2 : Choisir votre type de compte

Vous avez le choix entre deux types de comptes :

| Type | Description |
|------|-------------|
| **Visiteur** | Pour parcourir les créations et contacter les artisans |
| **Artisan** | Pour exposer et promouvoir vos propres créations |

### Étape 3 : Remplir le formulaire

Renseignez les informations suivantes :

- **Nom** (obligatoire)
- **Prénom** (obligatoire)
- **Email** (obligatoire) - sera votre identifiant de connexion
- **Mot de passe** (obligatoire) - minimum 8 caractères
- **Confirmation du mot de passe**

### Étape 4 : Valider

Cliquez sur **"Créer mon compte"**. Vous serez automatiquement connecté et redirigé vers votre espace personnel.

## 4. Se connecter

### Connexion utilisateur

1. Cliquez sur **"Connexion"** dans la barre de navigation
2. Assurez-vous d'être sur l'onglet **"Utilisateur"**
3. Entrez votre **email**
4. Entrez votre **mot de passe**
5. Cliquez sur **"Se connecter"**

### En cas de problème

- Vérifiez que l'email est correct
- Vérifiez la casse du mot de passe (majuscules/minuscules)
- Si vous avez oublié votre mot de passe, contactez l'administrateur

## 6. Parcourir les créations

### Accéder au catalogue

1. Cliquez sur **"Créations"** dans le menu
2. Vous arrivez sur la page listant toutes les créations disponibles

### Filtrer les créations

Utilisez les filtres disponibles pour affiner votre recherche :

- **Par catégorie** : Bijoux, Céramique, Sculpture, etc.
- **Par prix** : du moins cher au plus cher ou inversement
- **Par disponibilité** : articles en stock uniquement

### Affichage des créations

Chaque création est présentée sous forme de carte avec :
- L'image principale
- Le nom de la création
- Le prix
- Le nom de l'artisan
- Un bouton pour ajouter aux favoris (coeur)

### Accéder à la fiche produit

Cliquez sur une création pour voir sa fiche détaillée.

### Informations disponibles

La fiche produit affiche :

| Information | Description |
|-------------|-------------|
| **Images** | Photo principale et galerie |
| **Nom** | Titre de la création |
| **Prix** | Prix en euros |
| **Disponibilité** | En stock ou Indisponible |
| **Description** | Détail de l'oeuvre |
| **Matériaux** | Matières utilisées |

### Actions possibles

- **Ajouter aux favoris** : cliquez sur le coeur
- **Voir le profil de l'artisan** : cliquez sur le nom de l'artisan
- **Laisser un commentaire** : si vous êtes connecté

## 8. Gérer ses favoris

### Ajouter un article aux favoris

**Méthode 1 - Depuis le catalogue :**
1. Passez la souris sur une création
2. Cliquez sur l'icône coeur

**Méthode 2 - Depuis la fiche produit :**
1. Ouvrez la fiche de la création
2. Cliquez sur le bouton "Ajouter aux favoris"

### Consulter vos favoris

1. Connectez-vous à votre compte
2. Cliquez sur **"Mes Favoris"** dans le menu

### Page Mes Favoris

Cette page affiche :
- Le nombre total de favoris
- La valeur totale de votre sélection
- La liste de vos créations favorites avec :
  - L'image
  - Le nom
  - Le prix
  - Le nom de l'artisan
  - La disponibilité

### Retirer un favori

1. Accédez à **"Mes Favoris"**
2. Cliquez sur le bouton **"Retirer"** sous l'article concerné
3. L'article est immédiatement retiré de votre liste

## 12. Gestion des créations (Artisan)

### Accéder à la gestion

1. Connectez-vous avec un compte artisan
2. Allez dans **"Mon Compte"**
3. Cliquez sur **"Gérer mes créations"**

### Ajouter une nouvelle création

#### Étape 1 : Cliquer sur "Nouvelle création"

Un formulaire s'affiche avec tous les champs à remplir.

#### Étape 2 : Remplir les informations

| Champ | Obligatoire | Description |
|-------|-------------|-------------|
| Nom de la création | Oui | Titre de votre œuvre |
| Type d'objet | Oui | Ex: Vase, Collier, Tableau |
| Catégorie | Oui | Sélectionner dans la liste |
| Prix | Oui | En euros |
| Quantité | Oui | Stock disponible |
| Description | Oui | Détail de la création |
| Matériaux | Non | Matières utilisées |
| Dimensions | Non | Taille de l'objet |
| Couleur | Non | Couleur principale |
| Style | Non | Ex: Moderne, Traditionnel |

#### Étape 3 : Ajouter une image

1. Cliquez sur **"Choisir une image"**
2. Sélectionnez un fichier (JPG, PNG, GIF ou WEBP)
3. Taille maximum : 5 Mo
4. L'image sera redimensionnée automatiquement

#### Étape 4 : Valider

Cliquez sur **"Publier la création"**. Votre création sera immédiatement visible sur le site.

### Modifier une création

1. Dans la liste de vos créations, cliquez sur **"Modifier"**
2. Modifiez les champs souhaités
3. Cliquez sur **"Enregistrer les modifications"**

### Supprimer une création

1. Cliquez sur **"Supprimer"** sur la création concernée
2. Confirmez la suppression dans la fenêtre qui s'affiche

**Attention** : Cette action est irréversible.

### Mettre en avant une création

1. Cliquez sur **"Mettre en avant"** sur une création
2. Cette création apparaîtra en priorité sur votre profil public

## Conseils et bonnes pratiques

### Pour les acheteurs

1. **Créez un compte** pour profiter pleinement du site
2. **Utilisez les favoris** pour sauvegarder les créations qui vous plaisent
3. **Consultez les événements** pour rencontrer les artisans en personne
4. **Laissez des avis** pour aider les autres acheteurs

### Pour les artisans

1. **Soignez vos photos** : une belle image attire plus d'acheteurs
2. **Décrivez précisément** vos créations (matériaux, dimensions)
3. **Mettez à jour votre stock** régulièrement
4. **Complétez votre profil** avec votre histoire et vos réseaux sociaux
5. **Mettez en avant** vos meilleures créations
