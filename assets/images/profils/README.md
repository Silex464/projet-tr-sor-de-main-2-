# Dossier des images de profils

Ce dossier contient les photos de profil uploadées par les utilisateurs.

## Structure des fichiers
- Les photos sont nommées : `user_{id_utilisateur}_{timestamp}.{extension}`
- Formats acceptés : JPG, PNG, GIF, WEBP
- Taille maximale : 2 Mo

## Image par défaut
Si aucune image n'est définie, le système utilise une image générée ou un placeholder.
